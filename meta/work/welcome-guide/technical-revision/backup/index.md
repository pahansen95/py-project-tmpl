## Your Journey Starts Here

Welcome! This guide will help you get started with contributing to our Python project template. Whether you're new to Git, Python development, or just this project, we'll guide you through everything you need to know.

## 🏃 In a Hurry?

**[Jump to Quick Start](quickstart.md)** - Get from zero to pull request in 5 minutes!

## What You'll Learn

Through this guide, you'll gain practical experience with:

- **Core Concepts** - Understand the mental models and terminology we use
- **Environment Setup** - Configure your development environment with our automated tools
- **Development Workflow** - Learn the complete cycle from cloning to merging
- **Hands-on Practice** - Apply your knowledge through guided exercises
- **Quick Reference** - Access commands and troubleshooting tips

## Guide Structure

Each section builds on the previous one, creating a complete learning path:

1. **[Concepts](concepts.md)** - Mental models and shared vocabulary
2. **[Setup](setup.md)** - One-command environment configuration
3. **[Workflow](workflow.md)** - Step-by-step development process
4. **[Practice](practice.md)** - Hands-on exercises with validation
5. **[Reference](reference.md)** - Commands, tips, and troubleshooting

## Prerequisites

Before starting, ensure you have:

- Windows 11, macOS, or Linux
- Administrator access to install software
- Internet connection for downloading tools
- About 60 minutes for the complete guide

## Getting Help

If you encounter issues:

1. Check the [Reference](reference.md) page for troubleshooting
2. Review the [project documentation](../index.md)
3. Ask in project discussions
